## Instalación de dependencias 🔧

```
npm install o npm i
```

## Variables de entorno🚀
_Crear un archivo .env en la raiz y colocar los siguientes datos_

```
NODE_ENV=local
HOST=localhost
PORT=5000

//DB
DB_PORT=puertoDB
DB_NAME=nombreDB
DB_USER=usuarioDB
DB_PASSWORD=contrasenaDB
```

## Compilar proyecto en local 🚀

```
npm run dev
```
